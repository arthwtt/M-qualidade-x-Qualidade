## 📚 O que é Qualidade de Código?

Qualidade de código refere-se à capacidade do software ser:

- Legível  
- Manutenível  
- Reutilizável  
- Seguro  
- Escalável  
- Testável  
- Eficiente  
- Robusto  
- Padronizado  
- Bem documentado  

Um código de qualidade reduz erros, facilita manutenção futura e melhora a colaboração entre desenvolvedores.
